Power+ (dev)

Development Status

Active feature development - v3.1.0 (scheduled for stable release in January 2021)

To start using the dev version of Power+, visit Settings -> About -> Advanced Options and select "Test prerelease versions". Then, navigate to Settings -> Prerelease testing and select "Dev". Once you've done that, just reload Power+ to load the dev version. Keep in mind that the dev version uses the latest, often untested, code and can completely break. Use the dev version at your own risk.
