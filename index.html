Install Power+

Make sure the bookmarks bar is showing. If it's not showing, press Control + Shift + B to show it.

Drag this into your bookmarks bar to install Power+: Power+
Then, every time you visit PowerSchool, you need to click the Power+ bookmark to use Power+

code

Other versions of Power+

To load Power+ automatically at home, install the Power+ Chrome Extension.

To get the very latest, fresh-off-the-vine features, try the unstable dev version of Power+ by dragging this into your bookmarks bar: Power+ (dev)

To use Power+ on a non-chrome (not fully supported) browser, drag this into the bookmark bar: Power+
