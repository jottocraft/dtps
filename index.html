Privacy

  • Power+ does not collect any personal information
  • Power+ gets data directly from Canvas to your computer without being processed by a Power+ server or a 3rd party
  • Grades are calculated on-device
  • Prefrences and grade history are stored on-device
  • Power+'s code is open-source and hosted directly from GitHub
