Install Power+ on a d.tech-managed chromebook

The Power+ Chrome Extension is being blocked on d.tech chromebooks. You'll still be able to use Power+, it'll just be a bit harder to install and use. Follow these steps to install the Power+ bookmarklet.

  • Make sure you have the bookmarks bar enabled (if it's not, press Control + Shift + B to show it)
  • Drag this script into the bookmarks bar: Power+
  • Visit Canvas, then click on the bookmarklet to load Power+ (You have to be on Canvas for the bookmark to work)
devices

Install Power+ on your own device

  • Install the Power+ Chrome Extension
  • Visit Canvas, then click on the Power+ icon in the sidebar
Power+ App (beta)

Power+ is testing a new mobile grade calculator on Android. The Power+ Android app is still far from finished and is very much a work in progress. Make sure you understand the risks of using pre-release software before testing the Power+ App.

Privacy is a top priority for Power+. This summary is accurate as of November 14th, 2019.

General Information
person

Power+ is student-made

visibility_off

Power+ does not collect grades, classes, or assignments

devices

Power+ runs entirely on your device

storage

All of your data is processed on your device, not on a Power+ server

settings

code

group

Data Power+ collects
multiline_chart

Power+ anonymously collects basic information with Google Analytics. Data collected includes (but is not limited to):

  • IP Addresses
  • Basic device information
  • Basic usage statistics

Because data is collected anonymously, users cannot request or delete their Google Analytics data.

stars

For the small percentage of users that have manually opted into the PowerPoints rewards program and have accepted the risks, Power+ also collects:

  • Full name
  • Usage statistics
  • Points earned and other relevant information

If a user wishes to stop using PowerPoints, they can delete their data by clicking on the link at the bottom of the PowerPoints screen or by contacting me.

The privacy summary above is not a privacy policy or legal agreement. While I'll do my best to ensure that it is updated when necessary, I cannot guarentee that it will always be 100% accurate and up to date. The summary posted above is subject to change without notice. If you have questions or concerns about privacy in Power+, please contact me.
