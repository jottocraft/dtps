Install Power+

Get Power+ installed on your chromebook

Install the script

Drag and drop the Power+ link into the bookmarks bar: Power+

Run the script

You need to have PowerSchool open in order for the script to work. Open PowerSchool, then click on the Power+ bookmark. Follow any on-screen instructions to complete setup.


Other methods of installing Power+
  • To install Power+ at home, install the Power+ Chrome Extension
  • To install the dev (unstable) version of Power+, use this bookmark script
  • To run Power+ on Firefox (not supported), use this bookmark script
