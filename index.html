Install Power+

Get Power+ installed on your chromebook

Step 1: Copy the bookmark script

Click here to open a new tab with the bookmark script. Copy the contents of the page.

Step 2: Install the bookmark script

Press control + d on your chromebook. Click more when it tells you to make a new bookmark, and paste the bookmark script you copied in the URL box. You can set the name to whatever you want.

Run the bookmark script

You need to have PowerSchool open in order for the script to work. Open PowerSchool, then click on the bookmark script. Follow any on-screen instructions to complete setup.


Other methods of installing Power+
  • To install Power+ at home, install the Power+ Chrome Extension
  • To install the dev (unstable) version of Power+, use this bookmark script for Step 1
  • To run Power+ on Firefox (not supported), use this bookmark script
